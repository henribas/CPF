# CPF-CNPJ
Validação e geração de CPF e CNPJ com modelos sugeridos de classes e relacionamentos.

O Cadastro de Pessoa Física ou CPF é composto por 11 dígitos numéricos geralmente apresentados com uma máscara no formato: NNN.NNN.NNN-NN onde N representa os números do CPF. Para mais informações sobre o CPF o endereço da Wikipedia pode ser consultado: https://pt.wikipedia.org/wiki/Cadastro_de_Pessoas_F%C3%ADsicas
